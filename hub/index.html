Future hub

这里不是空白页，而是未来完整 OMS Web 的入口骨架。

账户、玩家主页、排行榜、谱面数据库、多人、观战、统计和服务入口都将在这里逐步落位。当前阶段先把页面层级、命名和跳转方向定住。

Server implications

未来入口的页面结构已经能反向约束后端边界。

  • 账户、聊天、好友和通知不能被压扁成单一 REST 入口。
  • BMS 的 `ruleset_id` 与 score 流程要先解决映射与 legacy gate 问题。
  • 谱面数据库和下载入口需要严格区分元数据层与大文件承载层。
