# Frontend Subline P1-A

`P1-A/` 是前端 `subline/` 下的首个功能方向目录，用作 Phase 1 的结构收束占位。

## 定位

- `P1` 表示当前仍处于 Phase 1：宣传展示阶段
- `A` 表示当前仅先建立第一个支线方向占位，用于验证新的支线收束方式
- 当前目录不代表已有开发成果，只代表后续前端支线内容应当如何归档

## 适用范围

- 可放置 Phase 1 中不直接进入主线、但与宣传展示站点有关的支撑性方向
- 例如视觉探索、内容补强、SEO、静态交互草案等
