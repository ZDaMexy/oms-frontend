# Frontend Subline P1-A

`P1-A/` 是前端 `subline/` 下的首个功能方向目录，当前用于承接 Phase 1 静态展示与宣传页骨架的首轮实现与内容确认工作。

## 定位

- `P1` 表示当前仍处于 Phase 1：宣传展示阶段
- `A` 表示当前采用的第一条宣传展示支线，用于收纳首版静态站骨架与内容确认节奏
- 当前目录已经不再是纯占位，而是首版静态宣传页的实际工作线

## 适用范围

- 可放置 Phase 1 中不直接进入主线、但与宣传展示站点有关的支撑性方向
- 例如首页结构、下载页、未来入口页、视觉探索、内容补强、SEO、静态交互草案等

## 当前状态

- 当前已完成首版静态宣传页骨架创建，并进入内容逐项确认阶段
- 当前支线成果已经回写主线进度文档
